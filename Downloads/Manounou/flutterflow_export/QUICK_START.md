# ⚡ Quick Start — Manounou

## 🚀 Lancer l'App en 3 Étapes

### 1. Vérifier l'environnement
```bash
cd flutterflow_export
flutter devices
```

### 2. Lancer sur votre plateforme préférée

**iOS Simulator** :
```bash
flutter run -d "iPhone 17 Pro"
```

**Web** :
```bash
flutter run -d chrome
```

**macOS** :
```bash
flutter run -d macos
```

### 3. C'est tout ! 🎉
